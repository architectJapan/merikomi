# merikomi（柱めり込み検討ツール）

木造住宅の「柱めり込み」を簡易に検討するためのHTMLツールと、説明書（HTML）を公開しています。

## 公開ページ
- 説明書（setumei.html）
  - https://architectjapan.github.io/merikomi/setumei.html

- ツール本体（merikomi_calc.html）
  - https://architectjapan.github.io/merikomi/merikomi_calc.html

## 使い方（最短）
1. 上の「説明書」を開く
2. 手順に沿って入力して結果を確認する

## 注意
- 本ツールの出力は参考値です。最終判断は設計者・審査機関の指示に従ってください。
